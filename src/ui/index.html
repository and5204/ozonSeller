Supply Manager

Автоматизированное создание поставок с подбором таймслотов и обработкой ошибок. Минимизирует ручную работу и повышает шанс успешной отгрузки.

Что делает система

Автоподбор таймслотов

Система находит ближайший доступный слот в заданном диапазоне времени.

Автоматические ретраи

При отсутствии слотов или rate limit запросы повторяются автоматически.

Обработка ошибок API

Система анализирует ошибки и пересоздаёт заявки при необходимости.

Непрерывная работа

Заявка выполняется в цикле до успешного результата или отмены.

Инструменты

Кроссдок

Поставка через точку отгрузки с контролем доступности таймслотов.

Direct поставка

Прямая поставка на склад с выбором FULL_FILLMENT складов.

Поиск складов

Поиск складов по названию и получение их параметров.

Документация

Подробное описание логики, ограничений и типовых ошибок.
